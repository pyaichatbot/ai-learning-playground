# Playground Information Architecture Refactor

## Top-Level Structure

AI Learning Playground
├── Basic Mode (Default)
│   ├── Existing modules
│   ├── Exploratory tools
│   └── Educational experiments
│
└── Advanced Mode
    ├── Prompt Reality Cockpit (v1)
    ├── Retrieval Reality Cockpit (future)
    ├── Cost Reality Cockpit (future)
    └── Agent Reality Cockpit (future)

---

## Navigation Rules
- No tabs inside cockpits
- Mode switch is explicit
- One cockpit active at a time

---

## Entry Experience
- First-time users land in Basic Mode
- Advanced Mode is discoverable, not forced
- Clear framing copy for Advanced Mode

---

## Growth Strategy
- Add cockpits, not features
- Each cockpit answers one systems question
- Retire experiments without affecting cockpits
