# User Story Seeds

## Prompt Reality Cockpit

- As an engineer, I want to paste a real production prompt so that I can see if it fits within context limits.
- As an architect, I want to see which parts of my prompt are likely ignored.
- As a tech lead, I want to understand prompt cost implications at scale.
- As a practitioner, I want to simulate truncation to understand failure modes.

---

## Advanced Mode

- As a returning user, I want a focused, opinionated interface so I can gain insight quickly.
- As a power user, I want system-level explanations instead of optimization tips.
- As a potential subscriber, I want advanced insights unavailable in basic mode.

---

## Basic Mode

- As a learner, I want open exploration without pressure.
- As a casual visitor, I want to try examples without commitment.
