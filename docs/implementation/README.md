## Note on Framework

The starter kit describes a Next.js architecture, but the actual project uses **Vite + React (SPA)**. The architectural principles remain the same, but implementation details differ.

See [SYSTEM-ARCHITECTURE.md](../architecture/SYSTEM-ARCHITECTURE.md) for the actual Vite implementation.
